# NursePlan Pro - Website

Professional nursing care plan management platform.

## Files

- `index.html` - Main landing page
- `404.html` - Error page
- `_config.yml` - GitHub Pages configuration
- `CNAME` - Custom domain
- `.nojekyll` - Disable Jekyll processing

## Deploy to GitHub Pages

1. Create a GitHub repository named `yourusername.github.io`
2. Push all files to the main branch
3. Site will be live at `https://yourusername.github.io`

## Custom Domain

1. Update `CNAME` file with your domain
2. Configure DNS records pointing to GitHub Pages
3. Site will be available at your custom domain
